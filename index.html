Why the name?




For people who are not familiar with computer science, my name seems kind of random. What do I mean? Just a dead cow? It's a little different from that...actually, a lot different. All of our computers have these little slots of memory which all have an address that is associated with it. These "memory addresses" are sometimes named with a 32-bit memory address; however, humans like to read these memory addresses in hexadecimal (base 16 numbering rather than base 10 which is what we normally use for numbers). If we divide 32 bits by 4 (because there are 4 binary numbers per 1 hex) we end with 8 hex numbers. The "letters" we can use for hex are A, B, C, D, E, and F. With those letters, we can rearrange these and create an 8 hex number memory address called DEADBEEF. What about the 0x? That is simply at the beginning of the address to denote it is using hexadecimal numbers. But why DEADBEEF? Why not some other words? Because it sounds awesome. That's why.
